SYNOPSIS
Description : This function is used to manage installation of apt packages on system. Var to set : aptPackageToInstall : Use this var to set wiche package to install with apt ${1} : Use this var to set [ aptPackageToInstall ] Base usage : Do_apt_install_package "My_package_to_install" Send Back : Installed package
