SYNOPSIS
Description : This function is used to manage uninstallation of apt packages on system. Var to set : aptPackageToUnInstall Use this var to set which package to uninstall ${1} : use this var to set aptPackageToUnInstall Basic use : Do_apt_install_package "My_package_to_uninstall" Send Back : Unsinstalled package
