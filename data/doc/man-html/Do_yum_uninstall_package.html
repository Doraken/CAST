SYNOPSIS
Description : This function is used to manage uninstallation of yum packages on system. Var to set : YumPackageToUnInstall Use this var to set which package to uninstall ${1} : use this var to set YumPackageToUnInstall Basic use : Do_yum_install_package "My_package_to_uninstall" Send Back : Unsinstalled package
