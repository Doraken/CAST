SYNOPSIS
Description : This function is used to manage installation of yum packages on system. Var to set : YumPackageToInstall : Use this var to set wiche package to install with yum ${1} : Use this var to set [ YumPackageToInstall ] Base usage : Do_yum_install_package "My_package_to_install" Send Back : Installed package
